# Zomato-Rating-Prediction-MLProject
Machine Learning project to predict restaurant ratings using NLP

This project predicts restaurant ratings using customer reviews and restaurant metadata from Zomato.

## Project Type
EDA + Regression + NLP

## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- NLP (Text Cleaning, TF-IDF)
- Machine Learning (Linear Regression, Random Forest)

## Project Workflow
1. Data cleaning and preprocessing
2. Exploratory Data Analysis
3. NLP on customer reviews
4. Feature engineering using TF-IDF
5. Model training and evaluation

## Evaluation Metric
- Mean Absolute Error (MAE)

## Result
The model achieved reasonable accuracy in predicting restaurant ratings.
